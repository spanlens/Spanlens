## Tone notes

- **HN**: Show HN ≠ marketing. Frame as "I built this because…". Honest about limitations.
- **r/LocalLLaMA**: technical, casual, Ollama angle. Avoid corporate voice.
- **r/MachineLearning**: research-tone. Lead with statistical methodology (Welch t-test, judge↔human correlation). Avoid promotion-flavored language — sub mods are strict.
- **awesome-list PRs**: minimal text, link + one-sentence description. No marketing.
- **dev.to / Medium**: developer-personal-experience tone. "We were burning $X/month and didn't know why" hook.
